أهداف الجلسة :

يتوقع من المتدرب بعد التدريب في الجلسة يكون قادراً على أن:

1- يتعرف على مفهوم لغة البرمجة CSS.

2- يُنشئ مجلد cssجديد للتصميم .


مفهوم لغة البرمجة CSS :
لغة CSS اختصار لـ Cascading Style Sheets أو أوراق الأنماط المتتالية ، هي لغة تنسيق (Style Sheet Language) تُستخدم لتصميم وتشكيل مظهر صفحات الويب المكتوبة بلغات ترميزية مثل HTML. بينما تقوم HTML ببناء الهيكل الأساسي للموقع (النصوص، الصور، الروابط)، تأتي CSS لتضفي الجمالية، الألوان، الخطوط، وتحديد أماكن العناصر (Layout) على الصفحة. أهمية لغة CSS:
تكمن أهمية CSS في تحويل صفحات الويب من نصوص بسيطة غير منسقة إلى واجهات مستخدم جذابة وسهلة الاستخدام، وتتلخص في النقاط التالية:
• الفصل بين المحتوى والتصميم: تتيح CSS فصل هيكل الصفحة (HTML) عن مظهرها، مما يسهل إدارة الموقع وتحديثه.
• تصميم متجاوب (Responsive Design): تمكّن المطورين من جعل المواقع تعمل وتظهر بشكل جيد على مختلف الشاشات والأجهزة (هواتف، أجهزة لوحية ).
• تحسين تجربة المستخدم (UX): تساهم في تنظيم العناصر وتنسيقها بشكل مريح للعين، مما يزيد من وقت بقاء الزائر في الموقع.
• توفير الوقت والجهد: يمكن كتابة كود CSS واحد وتطبيقه على مئات الصفحات، بدلاً من تنسيق كل صفحة على حدة.
• سرعة تحميل الموقع: استخدام ملفات CSS خارجية يقلل من حجم الكود، مما يسرع تحميل الصفحات.

نشاط (2) ما طريقة ربط ملف css بملف html ؟


إنشاء رابط بين ملف style.css و ملف html و ذلك باستخدام الوسم <link> داخل قسم <head> ثم حدد مسار ملف التصميم باستخدام خاصية hrefثم اكتب امر الربط في صفحة HTML
ذلك بوضع امر الربط بين وسمي <head>…</head> .يضع امر الربط بعد وسم <title>…</title> ، يستخدم ملف تصميم واحد للتحكم بتصميم العديد من ملفات HTML .
