أهداف الجلسة :

يتوقع المتدرب بعد التدريب في الجلسة أن يكون قادراً على أن :

1- يُحدد نوع الخط باستخدام خاصية font-family

2- يُحدد طراز الخطوط باستخدام خاصية font-style

3- يُحدد تباين الخطوط باستخدام خاصيةfont-variant

4- يُحدد وزن الخط باستخدام خاصية font-weight

5- يُحدد حجم الخط باستخدام خاصية font-size


نشاط (1) ما هي الخاصية المستخدمة لتحديد نوع الخط؟

تحديد نوع الخط باستخدام خاصية font-family ثم يكتب الفقرة في وسم <p>في ملفHTML . يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي { } خاصية font-family . يختار الخط المفضل من قائمة الخطوط المقترحة ثم يكتب علامة (; ) بعد تحديد الخاصية .


يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (2) كيف يم تحديد طراز الخطوط باستخدام خاصية font-style ؟

يكتب الفقرة في وسم <p>في ملف HTML . يكتب الوسم p{ }في صفحة css . يكتب داخل علامتي { } خاصيةFont-style . يحدد الطراز المفضل من القائمة المقترحة . يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (3) ما خطوات تحديد تباين الخطوط باستخدام خاصيةfont-variant؟

يكتب الفقرة في وسم<P> في ملف HTML ثم يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي { } خاصيةFont-variant . يحدد الطراز المفضل من القائمة المقترحة و يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (4) كيف تحدد وزن الخط باستخدام خاصية font-weight ؟

يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي { } خاصيةFont-weight. يحدد وزن الخط المفضل من القائمة المقترحة. يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .
