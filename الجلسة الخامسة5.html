أهداف الجلسة :

يتوقع من المتدرب بعد التدريب في الجلسة يكون قادراً على أن:

1. يُحدد إطار العنصر (سمكه ، نوعه، لونه) border

2. يُحدد التباعد الداخلي بين المحتوى و الإطارpadding

3. يُحدد الهوامش الخارجية للعنصر margin


نشاط (1) ما هي خطوات تحديد إطار العنصر (سمكه ، نوعه، لونه) border ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب بعد الوسم علامتي { }. يكتب داخل علامتي { } خاصيةborder. يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .



نشاط (2) كيف يتم تحديد التباعد الداخلي بين المحتوى و الإطار padding ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب بعد الوسم علامتي { }. يكتب داخل علامتي { } خاصيةpadding. يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .



نشاط (3) كيف يتم تحديد الهوامش الخارجية للعنصر margin ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب بعد الوسم علامتي { }. يكتب داخل علامتي { } خاصيةmargin. يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .
