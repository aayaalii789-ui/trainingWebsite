أهداف الجلسة :

يتوقع من المتدرب بعد التدريب في الجلسة يكون قادراً على أن:

1- يُحدد لون المقدمه باستخدام خاصية color في css

2- يُحدد لون الخلفية باستخدام خاصية background-color في css

3- يُحدد الصورة كخلفية background-image


نشاط (1) كيف نقوم بتحديد لون المقدمة باستخد ام لغة البرمجة CSS ؟

تحديد لون المقدمه باستخدام خاصية color في css يكتب الوسم <p>في ملفHTML ثم يكتب الوسم p {} في ملف css يكتب خاصية اللون في ملف css بين علامتي { }. يكتب علامة (; ) بعد تحديد الخاصية . .



نشاط (2) كيف نحدد لون الخلفية باستخد ام لغة البرمجة CSS ؟

تحديد لون الخلفية باستخدام خاصية background-color في css يكتب الوسم <p>في ملف HTML ثم يكتب الوسم p{ } المراد تفعيل خاصية لون الخلفيه عليه في ملف css . يحدد لون خلفية اي عنصر باستخدام خاصية background-color . يكتب علامة (; ) بعد تحديد الخاصية .

و بعدها يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق

نشاط (3) تحديد الصورة كخلفية background-image

يدرج الصورة في ملف الصور بمجلد HTML و يفعل خاصية background-image ثم يحدد مسار الصورة داخل الكود . يكتب علامة (; ) بعد تحديد الخاصية .
