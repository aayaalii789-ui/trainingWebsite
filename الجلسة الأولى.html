أهداف الجلسة :

يتوقع من المتدرب بعد التدريب في الجلسة يكون قادراً على أن:

1- يُعرف مفهوم تصميم موقع الويب .

2- يُحمل احد برامج تحرير النصوص visual studio code .

3- يُنشئ مجلد جديد Folder للمشروع باستخدام لغة البرمجة HTML .

4- يكتب الهيكل الاساسي لصفحة الويب باستخدام لغة البرمجة HTML.

5- يضيف عنوان لصفحة الويب باستخدام لغة البرمجة HTML .


ما هو تصميم موقع الويب ؟
تُعدّ مهارات تصميم مواقع الويب من المهارات الأساسية في عصر التكنولوجيا الرقمية، حيث يعتمد العالم بشكل متزايد على الإنترنت في مختلف المجالات.

ما هي لغة HTML؟
لغة HTML (HyperText Markup Language) اللبنة الأولى والأساسية في تصميم وبناء صفحات الويب، إذ تُستخدم لإنشاء هيكل الموقع وتنظيم محتواه مثل النصوص، الصور، الروابط، والجداول. هي لغة الترميز التي تصف بنية صفحات الويب تستخدم لبنية و محتوى صفحة الويب. و تستخدمم بشكل اساسي لتطوير المحتوى الأساسي لصفحة الويب تتمتع بدعم كبير يساعد على استخدام مناهج مختلفة لبنية صفحات الويب تعتمد مهارات تصميم الويب باستخدام HTML على فهم الوسوم (Tags) وكيفية استخدامها بشكل صحيح لبناء صفحات متناسقة وسهلة التصفح. كما تُسهم هذه المهارات في تحسين تجربة المستخدم من خلال تنظيم المحتوى بطريقة واضحة ومنهجية. ومع أن HTML لا تهتم بتنسيق الصفحات أو التفاعل المتقدم، إلا أنها تشكل الأساس الذي تُبنى عليه لغات وتقنيات أخرى مثل CSS وJavaScript. إن إتقان HTML يمكّن المصمم من إنشاء مواقع ويب بسيطة وفعالة، ويُعد خطوة أولى مهمة لكل من يرغب في دخول مجال تصميم وتطوير المواقع الإلكترونية.

- أولاً : أوامر لغة البرمجة :HTML
إنشاء صفحة ويب يتطلب استخدام أوامر لغة HTML، والتي تُعرف باسم "Tags". يقوم متصفح الإنترنت بقراءة وتنفيذ هذه الأوامر لتظهر تأثيراتها على صفحة الويب عند عرضها. تتميز بعض الأوامر بخصائص محددة، وكل خاصية يمكن أن تأخذ قيمة معينة. يتم تحديد هذه القيم داخل علامات التنصيص و لإنشاء صفحة ويب باستخدام لغة HTML، يجب اتباع الخطوات والقواعد الأساسية التالية:
- إنشاء ملف نصي باستخدام محرر نصوص بسيط من أجل كتابة وتعديل أوامر HTML كتابة أوامر HTML داخل علامات الزاوية (<>).
- معظم الأوامر في HTML تحتوي على بداية ونهاية، وتُغلَق بإضافة العلامة (>/ < ) في نهاية الأمر
- حفظ الملف النصي الذي يحتوي على أكواد HTML بامتداد html
- فتح الملف المحفوظ بواسطة مستعرض الإنترنت لعرض الصفحة والتحقق من النتائج النهائية للأوامر المكتوبة.
- يجدر الإشارة إلى أنه يمكن كتابة أوامر HTML باستخدام الحروف الكبيرة (Capital Letters) أو الصغيرة (Small Letters)، حيث لا يؤثر ذلك على وظيفتها.






ما هي خطوات تحميل برنامج محرر النصوص Visual Studio Code ؟

الانتقال إلى الموقع الرسمي لتحميل البرنامج بفتح Googleواذهب إلى موقع Visual Studio Code الرسمي ثم انقر Download حسب نظام التشغيل الخاص بك



تنزيل ملف التثبيت بعد الضغط على زر التحميل سيبدأ تنزيل ملف تثبيت VS Code على الكمبيوتر.
فتح ملف التثبيت وتشغيله افتح الملف الذي تم تنزيله لبدء معالج التثبيت.




ستظهر نافذة اتفاقية الترخيص، اضغط I accept ثم Next للمتابعة.
اختيار مكان التثبيت والخيارات اختر مكان تثبيت البرنامج، وأضف اختصارات مثل “Open with Code” إذا أردت، ثم اضغط install .



الانتهاء وتشغيل البرنامج بعد انتهاء التثبيت اضغط Finish وتشغيل VS Code لأول مرة.






كيف افتح مجلد جديد على برنامج محرر الأكواد ؟

ينشئ مجلد Folder جديد من new على سطح المكتب .


يفتح الملف في برنامج visual studio code من خلال File ثم نختار open folder
نختار المجلد ثم ينقر على زر select folder.





ينشئ ملف جديد داخل المجلد من New File
يسمي الملف الجديد باسم و إمتداد index.html










وضح كيفية كتابة الهيكل الاساسي لصفحة الويب باستخدام لغة ال HTML .


اولا اكتب الوسم <!DOCTYPE html> و هو الوسم المسئول عن ظهور الموقع في محركات البحث . ثم اكتب الوسم <html>....</html> و يكتب داخل هذا الوسم كل أكواد موقع الويب . قسم الهيكل الى قسمين <head>....</head > اكتب فيه عنوان موقع الويب . و <body> </body> اكتب داخله محتوى موقع الويب .



ما هو الوسم المستخدم في كتابة عنوان موقع الويب ؟


يكتب العنوان داخل الوسم
<title> ...</title>
يحفظ ما تم كتابته من خلال الضغط على قائمة file يختار Auto-save للحفظ التلقائي .
