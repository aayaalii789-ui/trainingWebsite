أهداف الجلسة :

يتوقع من المتدرب بعد التدريب في الجلسة يكون قادراً على أن:

1- يضع فراغ قبل أول سطر text-indent

2- يستخدم محاذاة النصtext-align

3- يضيف زخرفة النص text-decoration

4- يضيف المسافات بين الحروف letter-spacing

5- يحول النصtext-transform


نشاط (1) ما خطوات إضافة وضع فراغ قبل أول سطر text-indent ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي { } خاصية text-indent. يحدد الفراغ المفضل . يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .



نشاط (2) ما خطوات إستخدام محاذاة النصtext-align ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي خاصيةtext-align { }. يحدد المحاذاه المفضلة. يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (3) كيف يتم إضافة زخرفة النص text-decoration ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب بعد الوسم علامتي { } خاصية text-decoration. يحدد الزخرفة المفضلة . يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (4) كيف نقوم بإضافة المسافات بين الحروف letter-spacing ؟
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب بعد الوسم علامتي { } خاصية letter-spacing. يحدد المسافات بين الحروف المفضلة . يكتب علامة (; ) بعد تحديد الخاصية . يفتح صفحة الويب في المستعرض لعرض نتيجة التنسيق .


نشاط (5) وضح كيفية تحويل النصtext-transform .
يكتب الفقرة في وسم<P> في ملف HTML . يكتب الوسم p{ } في صفحة css . يكتب داخل علامتي { } خاصيةtext-transform. يحدد التحويل المفضل .

يستخدم القيمةlowercase لجعل كل الحروف صغيرة .
